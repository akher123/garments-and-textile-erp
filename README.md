# Garments and Textile ERP

The **Garments and Textile ERP** system is a comprehensive enterprise solution designed to streamline and automate business processes for the garments and textile industry.  
It provides integrated management of HR, Merchandising, Production, Planning, Accounting, Commercial, and Inventory operations within a unified web-based platform.

This ERP system is developed using **ASP.NET MVC 5** and **Entity Framework 6 (EF6)** with a modular architecture approach.  
It combines a responsive front-end powered by **jQuery**, **JavaScript**, **HTML**, and **CSS**, and includes advanced reporting via **SSRS (SQL Server Reporting Services)**.

The goal is to simplify complex garment and textile workflows through an efficient, scalable, and user-friendly solution.
